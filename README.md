# Console-Ticketing-System
A Python-based command-line interface (CLI) for managing bus and train ticket bookings, including user authentication and payment simulation. A lightweight, Python-based console application designed to handle basic bus and train ticket bookings. This project was developed to practice fundamental programming concepts including control flow, data structures, and modular function design.

## Features
* **User Authentication:** Simple login system for new and existing users.
* **Booking System:** Interactive selection of transport type (Bus/Train), routes, and seat availability.
* **Payment Simulation:** Basic payment gateway simulation with bill generation.
* **PNR Generation:** Randomized PNR/Ticket number generation for confirmed bookings.

## How to Run
1. Ensure you have [Python](https://www.python.org/) installed on your system.
2. Clone this repository: `git clone https://github.com/your-username/Console-Ticketing-System.git`
3. Navigate to the folder: `cd Console-Ticketing-System`
4. Run the script: `python main.py`
